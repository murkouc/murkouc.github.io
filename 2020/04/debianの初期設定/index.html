Debianの初期設定

Thu Apr 02, 2020 by Katsunori MIURA in  小樽商大  技術, debian

[基本環境]
  • VirtualBox 6.0.18 (Mac版)
  • Debian 10.3.0 (amd64)
[PATHへの/usr/sbin/の追加]
  1. cd /home/ユーザ名
    • ホームディレクトリに移動する
  2. touch .bash_profile
  3. vi .bash_profile
    • export PATH=$PATH:/usr/sbin/ と書いて保存する
  4. source .bash_profile
[一般ユーザに対するsudo権限の付与]
  1. su
  2. usermod -aG sudo ユーザ名
  3. reboot
[sources.listの編集]
  1. cd /etc/apt
  2. sudo vi sources.list
    • deb cdrom:[Debian GNU/Linux… の先頭に#を追加する
