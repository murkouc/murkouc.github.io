Docker環境の構築

Tue Mar 31, 2020 by Katsunori MIURA in  小樽商大  技術, docker, debian

[基本環境]
  • Debian 10.3.0 (amd64)
[Dockerのインストール]
  1. sudo apt update
  2. sudo apt install apt-transport-https
  3. sudo apt install ca-certificates
  4. sudo apt install curl
  5. sudo apt install gnupg2
  6. sudo apt install software-properties-common
  7. curl -fsSL https://download.docker.com/linux/debian/gpg | sudo apt-key add -
  8. sudo add-apt-repository “deb [arch=amd64] https://download.docker.com/linux/debian $(lsb_release -cs) stable”
  9. sudo apt update
  10. sudo apt install docker-ce
[Dockerの状態確認]
  1. sudo systemctl status docker
    • Active: active (running)なら動作中
  2. sudo docker ps -a
    • CONTAINER ID IMAGE…が表示されたらOK
[一般ユーザでDockerコマンドを使えるようにする]
  1. sudo usermod -aG docker ${USER}
  2. reboot
[Dockerコンテナの展開]
  1. docker pull debian:stable
    • Debianイメージをダウンロードしている
    • 他のディストリビューションでも良い
  2. docker run -itd --privileged -p 80:80/tcp -p 80:80/udp --name webserv debian:stable
    • ホストとコンテナの間で80ポートを繋いだコンテナを作る
  3. docker exec -it webserv /bin/bash
    • コンテナ内に入ったら、あとは、使うだけ
